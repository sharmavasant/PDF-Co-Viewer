
# PDF Co-Viewer

It is a collaborative pdf viewer.


## Run Locally

Clone the project

```bash
  git clone https://github.com/sharmavasant/PDF-Co-Viewer.git
```

Go to the project directory

```bash
  cd PDF-Co-Viewer
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

Node.js, Express.js, JavaScript, HTML, CSS
